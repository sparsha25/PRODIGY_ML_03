# Cat and Dog Image Classification using SVM

## 📌 Overview
This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs using the Kaggle dataset.

## 🚀 Features
- Image preprocessing
- Feature extraction
- Image classification
- Model evaluation

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## 📂 Dataset
Kaggle Cats and Dogs Dataset

## ▶️ How to Run
1. Install required libraries
2. Run the Python file:
bash
python main.py
